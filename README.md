# Movie-Database-app

A responsive Movie Database application built with React and Tailwind CSS that allows users to search for movies and view detailed information using the OMDB API.

## Features

- Search for movies
- View movie posters, title, and release year
- View detailed movie information
- Responsive design for mobile and desktop

## Tech Stack

- React (Vite)
- Tailwind CSS
- OMDB API
- GitHub
- Vercel (Deployment)
